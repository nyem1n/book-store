# 📚 도서 쇼핑몰 프론트엔드
이 프로젝트는 도서 API(book-shop)와 연동되는 도서 쇼핑몰 웹사이트입니다.
사용자는 도서를 검색하고, 상세 정보를 확인하고, 장바구니에 담거나 즐겨찾기할 수 있습니다.
